AliceChatBot
An AI-powered voice chatbot using LM Studio and custom TTS.
Chat with Alice through speech — talk and listen in real-time!

Features
Real-time voice conversation powered by LM Studio

Custom AI-generated TTS voice for Alice

Speech recognition for user input

Expressive anime-style personality

Requirements
Python 3.x

LM Studio (download separately)

Microphone and speakers/headphones

Python libraries (requirements.txt)
